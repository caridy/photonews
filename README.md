
photonews
=========

Photonews is an example web application showing "modown" in action.

## What's Next ?

- User interaction via pagination and search
- View composition

## Instructions

    npm i -g grunt (skip if you have `grunt` already installed globally)
    npm i
    grunt
    node ./app.js

## Application Structure

```
app_dir/
    ├── build/
    ├── lib/
    ├── public/
    ├── tests/
```
