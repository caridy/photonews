
photonews
=========

Example web application to show usage of "modown" family of components.

## What's Next ?

- ES6 modules
- view composition

## Instructions

    npm i -g grunt (skip if you have `grunt` already installed globally)
    npm i
    grunt
    node ./app.js

## Application Structure

```
app_dir/
├── build/ 
├── lib/
├── models/
├── public/
├── routes/
├── tests/
├── views/
```
